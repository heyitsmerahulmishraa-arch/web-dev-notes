## sum of numbers from 1 to n

![sum of numbers from 1 to n](../images/DSA/sum%20of%20numbers%20from%201%20to%20n.png)

```txt
input n
count = 1, sum = 0
while count <= n
    sum = sum + count
    count = count + 1
print sum
exit
```
