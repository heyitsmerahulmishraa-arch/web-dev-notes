# Node.js History

Node.js is an open-source, cross-platform JavaScript environment created to allow developers to build scalable network applications using JavaScript outside of a web browser.

# Key Historical Milestones

- **Inception (2009)**: Ryan Dahl created Node.js (initially named `web.js`) to address the concurrency limitations of the Apache HTTP Server. He combined Google's V8 engine with an event loop and low-level I/O API.

- **Launch of npm (2010)**: Isaac Schlueter introduced the Node Package Manager (npm), which revolutionized code sharing and allowed the ecosystem to grow rapidly.

- **Windows Support (2011)**: Joyent and Microsoft collaborated to realease a native Windows version, significatly expanding its user base beyond Linux and macOS.

- **The io.js Fork (2014)**: Frustrated by slow development under Joyent, Fedor Indutny and other contributors forked the project to create **io.js**, focusing on faster releases and open governance.

- **Reunification (2015)**: The two projects merged back together under the newly formed Node.js Foundation. This led to the release of Node.js v4.0, combining the stability of Node.js with the modern features of io.js.

- **OpenJS Foundation (2019)**: The Node.js Foundation merged with the JS Foundation to form the OpenJS Foundation, which currently oversees the project's development.

- **Modern Era (2020-Present)**: Recent versions (such as v20) have introduced advanced features like native TypeScript support, improved security permissions, and stable Fetch API integration.
